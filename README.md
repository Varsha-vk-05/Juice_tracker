# 🍹 Juice Tracker 

## 📌 Overview

**Juice Tracker** is an Android application developed to help users keep track of their favorite juices, including details such as flavor, description, color, and ratings. The app focuses on providing a clean and intuitive user interface along with smooth user interactions.

---

## ✨ Features

* 🧃 Add and manage different juice entries
* ⭐ Rate juices based on preference
* 🎨 Assign colors and descriptions to each juice
* 🧾 Clean and user-friendly UI
* ➕ Interactive bottom sheet for adding new items

---

## 🛠️ Tech Stack

* **Language:** Kotlin
* **UI Toolkit:** Jetpack Compose
* **Architecture:** MVVM (if applicable, edit if needed)
* **IDE:** Android Studio
